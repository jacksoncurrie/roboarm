# Roboarm
Website on a Raspberry Pi to control a robot arm.

## How to use
1. Change directory into PIGPIO
$ cd PIGPIO
2. Make PIGPIO
$ make
3. Install PIGPIO
$ sudo make install
4. Change directory into roboarm
$ cd ..
5. Install dependencies
$ sudo npm i
6. Start Roboarm
$ npm start
7. Open in browser
http://192.168.1.3:8080
